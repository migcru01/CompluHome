# "CompluHome"
Repositorio para el trabajo del grupo de la asignatura IS2 de Ing.Informática de la UCM.
Este trabajo es una continuación del proyecto *Compluhome* en IS1, todos los documentos de esta
pasada entrega se encuentran en esta [carpeta](DocumentosIS1/).

## Quienes somos
Compluhome es un proyecto, que busca ofrecer habitaciones a estudiantes de la UCM de manera asequible.
Buscamos conectar Estudiantes con Propietarios a través de la UCM, y así poder garantizar seguridad y 
precios justos.
